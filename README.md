* react-native-iconfont-cli

### Run
0. android studio 和 环境变量

1. 确保电脑上有一台开机的模拟器，如逍遥，夜神，然后链接 adb 端口

```shell
adb connect 127.0.0.1:21503 // 端口号百度
adb devices // 查看设备
```
2. 然后运行项目

```js
yarn install
yarn start
```
3. 在 expo 面板中点击 run in android 即可
